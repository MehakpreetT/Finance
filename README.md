Monte Carlo: 
This project is designed to estimate the future valuation of a stock portfolio with an intial investment, randomly weighted stocks and an X amount of test cases to accurately represent the findings. 
You can adjust the parameters to your liking to better understand the functionality of the process.

Option Pricing: 
This project helps you understand what Call and Put options really resemble and how decisions are made. 
The Black Scholes equation is used to symbolize the back bone of Quantative Finance. 
It also measures volatility to show which stocks are considered safer investments than the others.
Compare the Call and Put values to the current derivative prices and reflect if it is a feasible investment or not. 
I chose to use the TSX but feel free to use any stock from Yahoo Finance to see if it still follows through with the calculation.
